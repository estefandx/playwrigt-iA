# Playwright + TypeScript Base Project

This project is set up for end-to-end testing using Playwright and TypeScript.

## Estructura
- `tests/`: Contiene los escenarios de prueba.
- `playwright.config.ts`: Configuración de Playwright.

## Primer escenario
El archivo `tests/login.spec.ts` contiene un ejemplo de inicio de sesión exitoso en https://www.saucedemo.com/.

## Comandos útiles
- Ejecutar todas las pruebas:

```
npx playwright test
```

- Abrir el modo UI de Playwright:

```
npx playwright test --ui
```

## Más información
Consulta la documentación oficial de Playwright para más detalles: https://playwright.dev/
